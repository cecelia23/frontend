chapter one

In this section the proposed generalized block-size based prediction error embedding scheme is presented in details. The scheme predicts the values of the residual pixels in a block with generalized size using the Euclidean distance. The block generalization scheme presented here provides the flexibility of using blocks that have same dimensions for an image but their size vary arbitrarily rather than being 3×3 as used in all of the previous satellite pixel based block prediction methods. The problem of misleading block classifier has been resolved. These stated issues along with the message embedment and extraction rules are outlined at the following subsections.
